# Contributing

This repository is a local Chinese edition derived from BrightIntosh. Contributions should stay focused on local builds, Chinese UI polish, reliability, and macOS compatibility.

## Development

Use the local build script:

```sh
./scripts/build-local.sh
```

Before opening a pull request, make sure the project builds successfully and that `/Applications/BrightIntosh.app` launches.

## Scope

- Keep the UI native to macOS.
- Keep user-facing text concise and clear.
- Preserve GPL-3.0 licensing and upstream attribution.
- Avoid reintroducing App Store purchase flows, marketing links, or external author/social links into the local edition UI.
